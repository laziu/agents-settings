description: Implement the next task incrementally: build, test, verify, save to VCS.
---

Invoke `incremental-implementation` and `test-driven-development`.

For the next pending task:
1. Read acceptance criteria and relevant code
2. Write a failing test
3. Implement the minimum passing change
4. Run targeted tests, then relevant full checks
5. Run build/typecheck when applicable
6. Save the completed increment to the project VCS when requested or appropriate
7. Mark the task done

If a check fails, switch to `debugging-and-error-recovery`.
